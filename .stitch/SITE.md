# Comanda Pro Digital 2026 - Site Vision

Uma plataforma premium de autoatendimento e gestão para restaurantes, focada em velocidade, estética apetitosa e eficiência operacional.

## 1. Vision
Transformar a experiência de pedir comida em algo visualmente deslumbrante e sem atrito, integrando o cliente diretamente com a cozinha e o caixa através de um design "Crave-First".

## 2. Stitch Project Details
- **Project ID:** `5293229154161794662`
- **Design System:** Vibrant Gourmet (Vibrant Red, Deep Charcoal, Amber Accent)

## 3. Tech Stack
- **Frontend:** React + Vite + TailwindCSS (migrando para componentes Stitch)
- **Backend/DB:** Supabase (PostgreSQL + Real-time)
- **Auth:** Supabase Auth

## 4. Sitemap (Current & Planned)
- [x] **Login** (`login`) - Autenticação de clientes e funcionários.
- [x] **Menu Principal** (`menu`) - Navegação por categorias e lista de produtos.
- [x] **Carrinho** (`cart`) - Revisão de itens selecionados e observações.
- [x] **Checkout** (`checkout`) - Confirmação de pedido e seleção de mesa.
- [ ] **Kitchen Display System (KDS)** - Interface para a cozinha gerenciar pedidos em tempo real.
- [ ] **Waiter Dashboard** - Gestão de mesas e pedidos para garçons.
- [ ] **Cashier/POS** - Fluxo de fechamento de conta e pagamentos.
- [ ] **Admin Analytics** - Painel de controle com métricas de vendas e performance.

## 5. Roadmap
1. **Fase 1: Customer Flow (Completo)** - Refinar as telas de Menu, Carrinho e Checkout com o novo Design System.
2. **Fase 2: Kitchen Integration (Próximo)** - Criar o KDS com foco em status de pedidos e tempos de preparo.
3. **Fase 3: Service & Payments** - Implementar o Dashboard do Garçom e o Fluxo de Caixa.
4. **Fase 4: Management** - Criar painéis administrativos e analytics.

## 6. Creative Freedom / Ideas
- **Gamificação:** Badges para clientes frequentes.
- **Sugestões Inteligentes:** "Combina perfeitamente com..." no carrinho.
- **Visual tracking:** Animações fluidas mostrando o progresso do prato na cozinha.
